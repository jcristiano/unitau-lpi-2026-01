# Operadores de Incremento e Decremento

<Admonition type="info" title="Objetivo">
Aprender como funcionam os operadores de incremento e decremento em Portugol.
</Admonition>

## Conceito

Os operadores de incremento e decremento são utilizados para **aumentar ou diminuir o valor de uma variável em uma unidade**.

- Incremento: aumenta o valor em 1
- Decremento: diminui o valor em 1

## Operador de Incremento

```portugol
inteiro numero = 10
numero++
escreva(numero)
```

Resultado esperado:

```
11
```

## Operador de Decremento

```portugol
inteiro numero = 10
numero--
escreva(numero)
```

Resultado esperado:

```
9
```

Operador de Incremento ++

O operador ++ adiciona 1 unidade ao valor atual de uma variável inteira. É equivalente a escrever variavel = variavel + 1, porém de forma muito 

mais concisa.

Sintaxe

variavel++

Exemplo em Portugol

inteiro contador = 0

O operador é escrito após o nome da variável (pós-fixado).

contador++

escreva(contador)   // saída: 1

contador++

escreva(contador)   // saída: 2

contador++

escreva(contador)   // saída: 3

💡 Pense no ++ como um atalho elegante. Cada vez que for executado, a variável cresce 1.

Operador de Decremento --

O operador -- subtrai 1 unidade do valor atual de uma variável. É o oposto do incremento e funciona de maneira simétrica.

Sintaxe

variavel--

Exemplo em Portugol

inteiro vidas = 3

O operador é escrito após o nome da variável (pós-fixado).

escreva(vidas)   // saída: 3

vidas--

escreva(vidas)   // saída: 2

vidas--

escreva(vidas)   // saída: 1

🎮 Imagine um jogo onde o personagem perde uma vida a cada erro — o -- é exatamente isso!
