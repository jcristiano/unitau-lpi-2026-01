# 🧠 Pensamento Computacional

:::info O que é?

**Pensamento computacional** é a habilidade de resolver problemas de forma **lógica, estruturada e sistemática**, utilizando conceitos fundamentais da computação.

Ele não é exclusivo da programação — é uma forma de **organizar o raciocínio para solucionar problemas complexos**.

:::

---

# 🎯 Por que isso é importante?

Antes de escrever código, um desenvolvedor precisa saber:

- entender o problema
- dividir o problema em partes menores
- identificar padrões
- criar um processo para resolver o problema

:::tip

Programar é apenas **expressar em código uma solução que já foi pensada antes**.

:::

---

# ⚙️ Os 4 pilares do pensamento computacional

<div class="row">

<div class="col col--6">

## 1️⃣ Decomposição

Dividir um problema grande em **problemas menores**.

Exemplo:

Criar um sistema de notas pode ser dividido em:

- ler notas
- calcular média
- verificar aprovação
- mostrar resultado

</div>

<div class="col col--6">

## 2️⃣ Reconhecimento de Padrões

Identificar **similaridades entre problemas**.

Exemplo:

- cálculo de média
- cálculo de desconto
- cálculo de velocidade

Todos utilizam **operações matemáticas semelhantes**.

</div>

</div>

<div class="row">

<div class="col col--6">

## 3️⃣ Abstração

Focar apenas no **que é importante para resolver o problema**.

Exemplo:

Para calcular média de notas:

- precisamos das notas
- não precisamos do endereço do aluno

</div>

<div class="col col--6">

## 4️⃣ Algoritmos

Criar um **passo a passo para resolver o problema**.

Exemplo de algoritmo simples:

```
1. Ler nota 1
2. Ler nota 2
3. Calcular média
4. Mostrar resultado
```

</div>

</div>

# 💻 Exemplo prático

Problema: calcular média de duas notas.

### Pensamento computacional

1️⃣ Identificar os dados necessários  
2️⃣ Definir as operações  
3️⃣ Definir a ordem das etapas

### Algoritmo

```
leia nota1
leia nota2

media = (nota1 + nota2) / 2

escreva media
```

# 🚀 Conclusão

:::tip

O pensamento computacional é a **base de toda a programação**.

Antes de aprender linguagens como:

- Java
- Python
- C
- JavaScript

é fundamental aprender **a pensar como um solucionador de problemas**.

:::
