# Boas Práticas

## Nome das funções

Use nomes claros:
- calcularMedia
- verificarSenha
- exibirMenu

## Responsabilidade única

Cada função deve fazer apenas uma coisa.

## Validação

Sempre valide entradas:

- divisão por zero
- valores negativos
- entradas inválidas

## Organização

Divida seu código em funções pequenas.
