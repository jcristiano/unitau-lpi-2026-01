# Bibliotecas

## Biblioteca de Textos

```portugol
inclua biblioteca Textos --> t
```

Funções importantes:

- t.tamanho(texto)
- t.maiusculo(texto)
- t.minusculo(texto)
- t.posicao(texto, busca)
- t.substituir(texto, de, para)

## Biblioteca Matemática

```portugol
inclua biblioteca Matematica --> m
```

Funções:

- m.raiz(n, indice)
- m.potencia(base, exp)
- m.arredondar(valor, casas)
- m.absoluto(valor)

Constantes:

- m.PI
- m.E
