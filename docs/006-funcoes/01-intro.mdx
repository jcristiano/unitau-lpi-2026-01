# Funções em Portugol WebStudio

Este material tem como objetivo aprofundar o uso de funções no Portugol WebStudio, ajudando você a escrever programas mais organizados, reutilizáveis e fáceis de manter.

## O que você vai aprender

- Conceito de função
- Criação de funções
- Parâmetros e retorno
- Uso de bibliotecas
- Boas práticas
- Exercícios práticos

## Por que usar funções?

- Reutilização de código
- Organização
- Facilidade de manutenção
- Melhor leitura do código
