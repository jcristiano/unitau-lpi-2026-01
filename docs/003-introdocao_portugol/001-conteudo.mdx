# Introdução ao Portugol Webstudio

Aprender **algoritmos e lógica de programação** pode parecer difícil no
início.\
Por isso utilizamos uma ferramenta educacional chamada **Portugol
Webstudio**, que permite escrever programas utilizando uma linguagem
simples e próxima do português.

Essa abordagem ajuda o estudante a focar no **raciocínio lógico** e na
**estrutura dos algoritmos**, sem precisar se preocupar inicialmente com
detalhes complexos de linguagens profissionais.

------------------------------------------------------------------------

## O que é o Portugol Webstudio?

O **Portugol Webstudio** é um ambiente online criado para ensinar
programação de forma didática.

Ele permite:

-   Escrever algoritmos em **português estruturado**
-   Executar o código passo a passo
-   Visualizar a saída do programa
-   Testar ideias rapidamente
-   Aprender os fundamentos da lógica de programação

<Admonition type="info" title="Por que usar Portugol?">
Ele reduz a complexidade inicial da programação e permite focar no
desenvolvimento do **pensamento computacional**.
</Admonition>

------------------------------------------------------------------------

## Como acessar o Portugol Webstudio

O acesso é muito simples, pois a ferramenta funciona **diretamente no
navegador**.

### Passo 1 --- Acessar o site

Abra o navegador e acesse:

    https://portugol.dev/

### Passo 2 --- Abrir o ambiente

Ao entrar no site, você verá a interface do ambiente de programação.

Normalmente ela possui:

-   Área de **edição do código**
-   Botão para **executar o programa**
-   Área de **saída do programa**
-   Ferramentas para **novo arquivo e exemplos**

<Admonition type="tip" title="Dica">
Não é necessário instalar nada no computador. Basta ter acesso à internet e um navegador.
</Admonition>

------------------------------------------------------------------------

## Conhecendo a Interface

O ambiente do Portugol Webstudio é dividido em algumas partes
principais.

### Editor de Código

É onde você escreve seus algoritmos.

Exemplo:

```portugol
programa {
  funcao inicio() {
    escreva("Olá, mundo!")
  }
}
```

### Botão Executar

Depois de escrever o algoritmo, você pode clicar em **Executar** para
ver o resultado.

### Área de Saída

Mostra o resultado do programa.

No exemplo anterior, o resultado seria:

    Olá, mundo!

## Como o Portugol ajuda no Pensamento Computacional

O **pensamento computacional** é a habilidade de resolver problemas de
forma lógica e estruturada.

O Portugol ajuda nisso porque permite praticar:

### 1. Decomposição

Dividir um problema grande em partes menores.

Exemplo:

Problema: calcular a média de notas.

Passos:

1.  Ler a primeira nota
2.  Ler a segunda nota
3.  Somar as notas
4.  Dividir por 2
5.  Mostrar o resultado

------------------------------------------------------------------------

### 2. Sequência lógica

Os comandos são executados **na ordem em que aparecem**.

``` portugol
programa {
  funcao inicio() {
    escreva("Passo 1")
    escreva("Passo 2")
    escreva("Passo 3")
  }
}
```

------------------------------------------------------------------------

### 3. Abstração

Focar apenas nas informações importantes do problema.

Por exemplo:

Para calcular a média, não precisamos saber:

-   nome do aluno
-   idade
-   endereço

Precisamos apenas das **notas**.

1.  Acesse o Portugol Webstudio

3.  Utilize o comando `escreva`
4.  Execute o programa
